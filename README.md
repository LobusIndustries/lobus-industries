# Lobus Industries

Marketing site for Lobus Industries — we build professional websites for service-based businesses at no upfront cost. Customers only pay $100/month for hosting.

Built with Next.js 16 + Tailwind v4. Deployed on Vercel.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Pushes to `main` deploy automatically via Vercel.
