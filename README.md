## What Lobus Industries does

Lobus Industries builds custom websites for service-based small businesses — plumbers, electricians, landscapers, HVAC contractors, cleaners, tutors, trainers, mechanics, photographers, bookkeepers, and similar trades and pro services.

Instead of charging a large upfront build fee like most agencies, Lobus builds the site for free and earns revenue through a flat **$100/month** hosting and maintenance subscription.

- **$0** to build
- **$100/month** flat for hosting, SSL, backups, and ongoing content updates
- **~1 week** from kickoff to review-ready
- **No contract.** Cancel any time. Customer owns the domain.

## How the site is built

- **[Next.js 16](https://nextjs.org)** (App Router) + **[Tailwind v4](https://tailwindcss.com)**
- Fully static, edge-hosted on **[Vercel](https://vercel.com)**
- Dynamic Open Graph images via `next/og`
- JSON-LD structured data (Organization, ProfessionalService, FAQPage)
- Sitemap, robots, and `llms.txt` for both traditional and AI search
- Contact form wired to **[Formspree](https://formspree.io)**
- Analytics and Speed Insights via Vercel

## Develop locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Every push to `main` auto-deploys to production via Vercel.
